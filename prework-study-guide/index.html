HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • In this list all "" should be replaced with <>
  • "!DOCTYPE html" declaration tells the browser what document type it should expect to open. In this case, html
  • Declarations are not case sensitive, "HTML" elements are case sensitive
  • "HTML" attributes define additional information about an element
  • Example of attribute: lang="en", which tells the browser which language to use for the webpage, in this case English. *Attributes are always defined at the beginning of an element, never at the closing.
  • The "style" element defines the CSS styles associated withe a website
  • The "base" element defines the base URL for a webpage
  • The "link" element connnects an external resource to the HTML document, like the CSS code we linked to this webpage
  • The "meta" element defines metadata such as character set, description, keywords, author, and viewport
  • The "script" element embeds client-side scripts in the HTML document, such as adding JavaScript to the website
  • The "charset" element sets our character encoding
  • The "http-equiv" element sets an HTTP header for our content
  • The "viewport" element sets the default view of our webpage
  • The "ul", "ol", and "li" elements represent unordered lists, ordered list, and list items

CSS

  • A margin indicates how much space we want around the outside of an element
  • A paddding indicates how much space we want around the content inside an element
  • Declerations contain two important components: the CSS property we want to apply and the value of the property
  • Example: color: blue; includes CSS property "color" and value of the property "blue"
  • To apply a rule to all the elements visible on a page, use the * symbol preceding {}
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • A period preceeding an attribute will designate it as a class selector. Any HTML element assigned to the class attribute will gain the property assigned from the CSS

Git

  • Add notes for git
  • git status: checks what branch we are currently on
  • git checkout -b "branch-name": creates a new branch and switches to it
  • git checkout main: switches to the main branch or working code
  • git add -A: The "add" command adds modifications in the current working branch to the staging area. The "-A" flad indicates that we want to add all the changes
  • git commit -m "description of what will be added": commits all updates to working branch
  • git pull origin main: this command will pull a branch's modifications into the local environment. In this case the "main" branch was defined since the word main was used
  • git push origin "branch created": this command will push all local changes to the remote GitHub branch
  • A pull request is used to merge a comparison branch into the main branch
  • The base branch is where the changes will be merged into.
  • When a pull request is open, team members can review your code to make sure it looks good and funtions before merging into main branch.

JavaScript

  • In JavaScript, we declare a variable by using the word var, after var we give the variable a name
  • After the variable is named we use an assignment operator = to assign specific data that we want stored in our variable
  • console.log method outputs a message to the web console by adding an argument
  • Each ling of JavaScript must end in a semicolon
  • All string variables are surrounded by quotation marks
  • An "if" statement is a way of introducing decision-making into our code
  • When the computer encounters an "if" statement, it will check to see if the condition of the code is truthy. If it is, it will run the code provided. If the condition is not truthy, the computer will skip the code
  • JavaScript "if" statements can also be accompanied by "else if" statements if more than one condition is wanted
  • An array is a single variable that is used to hold a group of data, typically to hold data that is related in some way
  • "for" loops use a predictable patter of indices to perform a task on all items in an array by allowing a single block to be executed over and over
  • Developers must use another statement, such as the conditional "if" statement, to interrupt the control flow by executing a block of the code a specific number of times
  • You can define the length of an array by using the "length" command
  • A function is a set of instructions that tells the computer how to perform a task
  • To declare a function, use the "function" key word, followed by its unique name in parentheses. Then add "{}" that contain the code for the task you want to perform
